Expense Tracker — 

Overview
This is a simple Java-based Expense Tracker that allows users to record, view, and summarize their daily expenses. It runs in the console and is designed for quick and easy financial tracking.

Features:-
- Add expenses with category and amount
- View all recorded expenses
- Calculate total spending
- Summarize expenses by category
- Input validation for cleaner data
- Auto timestamping using LocalDate

Requirements:-
- Java 8 or higher
- Any IDE or terminal that supports Java compilation

 How to Run:-
- Clone or download the project files.
- Open your terminal or IDE.
- Compile the code:
Bash:-
javac ExpenseTracker.java
- Run the app:
Bash:-
java ExpenseTracker

Usage:-
When prompted:
- Enter category: Type a label like Food, Rent, Travel, etc.
- Enter amount: Type the expense amount in ₹ (e.g., 250.50)
- Use the menu to view expenses, see totals, or exit.

Future Enhancements:-
- Save/load expenses to a file
- Monthly summaries
- GUI version using JavaFX or Swing
- Export to CSV or PDF
